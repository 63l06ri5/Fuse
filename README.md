## Introduction

Fuse.js is a lightweight fuzzy-search, in JavaScript, with zero dependencies.

## Documentation

To checkout out live examples and docs, visit [fusejs.io](https://fusejs.io).

## Installation

**NPM**

```sh
$ npm install --save fuse.js
```

**Yarn**

```sh
$ yarn add fuse.js
```
